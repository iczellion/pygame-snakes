# pygame-snakes

A simple Snake game written using Python and the pygame library.  You can control it or let the machine learn to control it...


# Requirements

- Python 3.11.x
- Poetry 1.8+


# Develop locally

```shell
# Create virtual env
poetry env use $(which python3.11)

# Activate virtualenv
poetry shell

# Start snake
python src/program.py -h
```
