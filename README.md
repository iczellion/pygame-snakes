# pygame-snakes

A simple Snake game written using Python and the pygame library.  You can control it or let the machine learn to control it...


# Requirements

- Windows 10+
- Python 3.9.x


# Develop locally

```powershell
# Create virtual env
python3 -m venv .\venv

# Activate virtualenv
.\venv\Scripts\activate

# Install pip dependancies
pip install -r .\requirements.txt
```
